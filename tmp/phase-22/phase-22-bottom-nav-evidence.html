Desktop에서는 MobileAppShell과 BottomNavigation이 max-w-2xl로 중앙 정렬됩니다. 하단 nav는 fixed이지만 내부 grid는 shell과 같은 width cap을 사용하고, content는 pb-[var(--qling-space-scroll-bottom)]로 마지막 카드/CTA가 가려지지 않도록 합니다.

PWA/브라우저 수동 노트

환경노트
iOS safe-area실기기 검증은 수행하지 않았습니다. CSS는 env(safe-area-inset-bottom, 0px)를 --qling-space-safe-bottom에 중앙화하고, BottomNavigation paddingBottom과 MobileAppShell scroll bottom 모두 같은 변수를 사용합니다. fake home indicator는 추가하지 않았습니다.
Android Chrome/PWA실기기 검증은 수행하지 않았습니다. Android/PWA에서도 safe-area fallback은 0px이며 nav height 5rem + 2rem scroll clearance가 적용됩니다. PWA install/share 자체 검증은 Phase 23/24 범위입니다.
Desktop browser실브라우저 스크린샷은 생성하지 않았습니다. 코드상 fixed nav와 main max width가 모두 max-w-2xl 기준이고, build가 통과했습니다.
